# TAGS MODULE

## Purpose

This module is part of our AWS foundational resources, and will be used by almost every other module this project.  The structure of this module is intentionally consistent and every future module should follow the same pattern unless there is a compelling reason not to.

## Inputs

Input files include:

- versions.tf
- variables.tf
- locals.tf

## Outputs

* outputs.tf

## Example Usage

* Usage examples can be found in tags/examples directory
